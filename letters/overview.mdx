## Generating Letters with the Letters Feature
Letters allows you to create custom documents from core details, transcriptions, and typed or scanned PDFs.

### Key Features:
- **Customizable Templates**: Create or adjust templates based on your style by submitting examples.
- **Version History**: Track changes with a comprehensive version history.
- **Request Changes**: Easily request edits to generated letters.
- **File Integration**: Attach typed or scanned PDFs for inclusion in your final letter.
- **Transcriptions**: Attach transcriptions that have been generated in Letters.
