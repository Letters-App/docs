## Chat Feature Overview

Use **Chat** to ask questions, generate, or manipulate text efficiently. Medical staff can draft practice materials, while doctors frequently convert consultation notes into letters. 

### Key Features:
- **File Attachments**
- **Speech-to-Text**
- **Text-to-Speech**
- **Customisable Memory**
- **Voice Adjustments**
- **Shortcuts**
