## Customise Your Documents

Customisation allows you to tailor aspects of your workflow, such as letter templates, transcription accuracy, and Assistant functionalities. Hereâ€™s how you can customise different areas:

<AccordionGroup>

  <Accordion title="Letters" icon="envelope">
    * **Letter Types**: Create and manage your letter types.
    * **Requested Edits**: Save requested edits for future letters.
    * **Weight & Height Field + BMI**: Add weight and height fields to letter setup, with automatic BMI calculation.
    * **Details Section**: Add optional name, pronoun, and date fields to letter setup.
  </Accordion>

  <Accordion title="Transcribe" icon="microphone">
    * **Dictionary**: Add custom words to improve transcription accuracy.
    * **Intelligence**: Enable automatic summarisation and analysis of transcriptions.
    * **Your Notes Section**: Add space to take notes during transcription, incorporating into Intelligence & Letters.
    * **Skip Transcript (Intelligence only)**: Process audio directly into Intelligence for faster output.
    * **Show Consent Pop-up**: A prompt at the start of each recording to request patient consent.
    * **Remove Casual Conversation**: Exclude casual, non-medical speech from transcriptions.
  </Accordion>

  <Accordion title="Assistant" icon="message">
    * **Shortcuts**: Create shortcuts for frequent requests to save time.
    * **Memory**: Customise memory for future conversations.
    * **Voice**: Change the voice used for spoken messages.
  </Accordion>

</AccordionGroup>
